# README #

Este Projeto é destinado aos testes de API da aplicação Sicredi.


### Technologies ###

* Java
* Cucumber
* Serenity
* JUnit


### Dependencies ###

* Java
* Maven | Grandle


### Usage ###

To run all tests 
```
  $ mvn clean verify
```

### Usage ###

To get reports, acess
```
  $ ./target/site/serenity/index.html
```
